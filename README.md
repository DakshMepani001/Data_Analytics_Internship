# Data Analytics Internship at SkyBrisk

## Overview

This repository contains all the work completed during my **Data Analytics Internship at SkyBrisk**. The internship focused on developing practical skills in Python programming, data analysis, data cleaning, data manipulation, and data visualization.

Throughout the internship, I completed weekly theory modules, hands-on projects, and client projects that helped me understand the complete data analytics workflow. The projects involved working with Python, NumPy, Pandas, Matplotlib, and Seaborn to analyze data, generate insights, and create visual dashboards.
